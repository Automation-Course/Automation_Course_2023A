Read Me:
In Order to use the calculator - 

1. Binary To Decimal:
Insert the desire number with 0 and 1 digits only, no negative
press "Binary To Decimal" button
The converted number will appear on the screen.

2. Decimal To Binary:
Insert the desire number with 0-9 digits only, no negative
press "Decimal To Binary"
The converted number will appear on the screen.
