Download the program  to the Arduino controller, connect the controller to a voltage of five volts.

Expected system operations: 
1.	When no signal has been give, the light will be red and the screen will show “No passage”
2.	When you open regularly using IR there will be 2 sec delay, afterwards the gate will open,the light will be green and screen will show “pass” for 5 sec, then will close the gate
3.	When you open  using Button there will be no delay,  the gate will open,the light will be green and screen will show “Emergency” for 5 sec, then will close the gate.
